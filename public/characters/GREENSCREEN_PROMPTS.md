# Cast green-screen prompts (for clean cut-outs)  ·  Nano Banana 2

Same clean-cut recipe as `BLU_GREENSCREEN_PROMPT.md`, for the rest of the cast. Generate
each on a flat green screen, send me the file path, and I'll key it out in Python
(`rembg` + green-spill suppression) and wire it in — exactly like the hero Blu.

**Shared for every prompt (don't drop these):**
- Upload the matching reference sheet from `blubabyshow/character/finalcharacters/`.
- **Environment:** one flat, uniform **chroma-green screen** (`#00B140`), edge to edge — no
  gradient, no props, **no cast shadow on the floor**.
- **Composition:** full body, centred, **generous margin so the character never touches an
  edge**; eye-level, slight 3/4 angle, 50mm look.
- **Lighting:** soft, even, flat studio light, **no harsh shadow, no dark edge rim**.
- **Style:** high-fidelity 3D Pixar / Illumination render, soft rounded forms,
  subsurface-scattering skin, big expressive eyes, smooth clean surfaces.
- **Avoid (every prompt):** any green on the character, cast shadow, gradient/textured
  background, character touching the frame edge, text, watermark, extra limbs, harsh
  shadows, dark outline/rim.
- ⚠️ None of our cast wears green, so they're all green-safe.
